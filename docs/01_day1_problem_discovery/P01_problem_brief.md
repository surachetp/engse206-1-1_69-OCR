		
P01 — Problem Brief		
		
Problem statement		
		
ปัจจุบันพนักงานหอพักต้องเดินจดบันทึกมิเตอร์ไฟประจำเดือนด้วยการเขียนลงกระดาษและนำกลับมาคีย์ลงคอมพิวเตอร์ด้วยตนเอง ซึ่งการจดบันทึกผ่านหลายขั้นตอนแบบแมนนวลนี้ ทำให้เกิดปัญหาลายมืออ่านยาก ข้อมูลไม่เป็นมาตรฐาน มีโอกาสเกิดความผิดพลาดในการจดบันทึกตัวเลข (Human Error) หรือคีย์เลขสลับกัน ส่งผลให้เกิดความล่าช้าในการออกบิล และเกิดข้อพิพาทกับผู้เช่าห้องพักจากการคำนวณค่าไฟที่ผิดพลาด		
		
Target outcome		
		
มีระบบโมบายแอปพลิเคชันสำหรับพนักงานจดมิเตอร์ ที่ใช้เทคโนโลยี OCR ในการสแกนและอ่านค่าจากมิเตอร์ไฟแบบจานหมุน (Analog Meter) พร้อมระบบจัดการหลังบ้าน (Web Admin) สำหรับตรวจสอบ ยืนยันข้อมูล และออกบิลค่าไฟได้อย่างถูกต้อง รวดเร็ว และเป็นระบบกลางที่ตรวจสอบย้อนหลังได้		
		
Business/User Goals		
		
Business Goals		
		
ลดอัตราความผิดพลาดในการออกบิลห้องพักจากปัญหาคีย์เลขมิเตอร์ผิดให้เหลือ 0%		
		
ลดระยะเวลาและขั้นตอนการทำงานของพนักงานในการเดินจดและคีย์ข้อมูลลงระบบ		
		
มีฐานข้อมูลประวัติการใช้ไฟฟ้าที่ถูกต้องรวดเร็ว เพื่อนำไปใช้ในการวิเคราะห์และบริหารจัดการพลังงานภายในหอพัก		
		
User Goals		
		
พนักงานสามารถบันทึกเลขมิเตอร์ได้ง่ายผ่านมือถือ โดยไม่ต้องพกกระดาษปากกาและไม่ต้องพิมพ์ตัวเลขเองทั้งหมด		
		
มีระบบช่วยแจ้งเตือน (Alert) หน้างานทันทีหากตัวเลขที่สแกนได้มีความผิดปกติ (เช่น ค่าน้อยกว่าเดือนก่อน)		
		
ผู้บริหาร/เจ้าของหอพักสามารถตรวจสอบความโปร่งใสและดูภาพหลักฐานรูปถ่ายมิเตอร์ย้อนหลังได้เมื่อผู้เช่ามีข้อสงสัย		
		
Primary Users		
		
User	Goal	Pain point
พนักงานจดมิเตอร์ (Meter Reader)	บันทึกเลขมิเตอร์ได้รวดเร็ว ถูกต้อง และส่งข้อมูลเข้าส่วนกลางได้ทันทีจากหน้างาน	ฝาครอบมิเตอร์สะท้อนแสงไฟแฟลชบังตัวเลข, ตัวเลขจานหมุนก้ำกึ่งเลื่อนหลักทำให้อ่านยาก, เสียเวลาคีย์ข้อมูลซ้ำซ้อน
เจ้าหน้าที่ฝ่ายบัญชี/ออกบิล (Billing Officer)	ตรวจสอบข้อมูลมิเตอร์และออกบิลค่าไฟได้อย่างถูกต้องตามรอบเวลา	ได้รับข้อมูลล่าช้า, ลายมือพนักงานอ่านยาก, ต้องเสียเวลาคอยแก้ไขบิลย้อนหลังเมื่อเกิดการจดผิด
ผู้บริหาร / เจ้าของหอพัก (Owner)	ติดตามภาพรวมรายได้ค่าไฟ และตรวจสอบความโปร่งใสในการจดบันทึกได้ง่าย	ไม่มีหลักฐานรูปถ่ายหน้าปัดมิเตอร์ยืนยันเมื่อผู้เช่าร้องเรียนว่าค่าไฟแพงเกินจริง
Success Indicators		
		
พนักงานสามารถสแกนและยืนยันเลขมิเตอร์เสร็จสิ้นภายใน 3 คลิกต่อหนึ่งห้อง		
		
โมเดล OCR สามารถสแกนอ่านค่ามิเตอร์จานหมุนได้ถูกต้องไม่ต่ำกว่า 95% (โดยมีกลไกให้มนุษย์กดยืนยัน/แก้ไขก่อนบันทึก)		
		
ระยะเวลาตั้งแต่เริ่มเดินจดจนถึงพร้อมออกบิลลดลงอย่างน้อย 50% เมื่อเทียบกับวิธีเดิม		
		
อัตราการร้องเรียนเรื่องบิลค่าไฟผิดพลาดเนื่องจากการจดเลขมิเตอร์ลดลงจนเป็นศูนย์		
		
Assumptions & Hypothesis		
		
Assumption / Hypothesis	Impact if wrong	How to verify
พนักงานสามารถเปิดแฟลชถ่ายรูปมิเตอร์จานหมุนได้โดยไม่มีแสงสะท้อนบังตัวเลข	โมเดล OCR อ่านค่าไม่ได้เนื่องจากเกิดจุดบอดสีขาวบนภาพถ่าย	ทดลองใช้มือถือถ่ายมุมเอียง 15-30 องศาหน้างานจริง แล้วทดสอบกับ OCR Tool เพื่อดูความแม่นยำในการอ่าน
บริเวณตู้มิเตอร์ไฟฟ้าตามทางเดินหอพักมีสัญญาณอินเทอร์เน็ตเสถียร	แอปพลิเคชันค้าง ไม่สามารถส่งรูปไปประมวลผลบน Server ได้	เดินเช็กสัญญาณเน็ตตามซอกตึก/ชั้นต่าง ๆ หากไม่มี ต้องเปลี่ยนสถาปัตยกรรมแอปเป็น Offline-first (เซฟลงเครื่องก่อนแล้วค่อยซิงค์)
ค่ามิเตอร์ที่อ่านได้จากระบบ OCR จะต้องเข้าเงื่อนไข Validation Logic ทุกครั้ง	ระบบปล่อยผ่านข้อมูลที่ผิดพลาด เช่น ค่าน้อยกว่าเดือนก่อน หรือหน่วยกระโดดเกินจริง	ออกแบบ Decision Matrix ร่วมกับเจ้าหน้าที่หอพัก เพื่อกำหนดเกณฑ์ Business Rules ในการแจ้งเตือนพนักงานหน้างาน
พนักงานจดมิเตอร์ทุกคนใช้สมาร์ทโฟนที่รองรับฟีเจอร์กล้องและการประมวลผลภาพได้ดี	ตัวแอปทำงานช้าหรือกล้องไม่โฟกัส ทำให้ OCR ทำงานผิดพลาด	สำรวจรุ่นมือถือที่พนักงานใช้งานอยู่จริงเพื่อกำหนด Minimum Specification ของแอปพลิเคชัน
